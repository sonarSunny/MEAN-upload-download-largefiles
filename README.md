# Upload-download-large-JSON-file-using-MEAN
  Upload and download 142 MB json file in MongoDB using node js express and Angular-8


# Prerequisite
  >Angular-8
  >node.js
  >mongodb
  >express js
# Set-UP
  > Backend setup
  ```
    > cd backend
    > npm install 
    > node index.js 
  ```
  > Frontend Setup
  ```
    > cd frontend
    > npm install
    > ng serve
  ```
